1. Introduction to C++

What is C++?

C++ is a general-purpose, compiled programming language developed by Bjarne Stroustrup at Bell Labs in 1979. Originally called "C with Classes," it was renamed to C++ in 1983. C++ is an extension of the C programming language with object-oriented programming (OOP) features, making it a multi-paradigm language that supports procedural, object-oriented, and generic programming.

C++ is a compiled language, meaning source code is translated into machine code by a compiler before execution. This results in high-performance applications. C++ provides both high-level abstractions and low-level memory control, making it suitable for system programming, game development, embedded systems, and performance-critical applications.

Key Characteristics of C++:

  • Object-Oriented Programming: Supports classes, objects, inheritance, polymorphism, and encapsulation
  • Compiled Language: Source code is compiled to machine code for fast execution
  • Memory Management: Direct control over memory through pointers and manual memory management
  • STL (Standard Template Library): Rich collection of template classes and functions
  • Templates: Support for generic programming and code reuse
  • Multi-paradigm: Supports procedural, OOP, and generic programming
  • Performance: Close to hardware with minimal runtime overhead
  • Portable: Can be compiled on various platforms

Why Learn C++?

  • Industry Standard: Widely used in game development, system programming, and embedded systems
  • Performance: One of the fastest programming languages, ideal for performance-critical applications
  • Foundation for Other Languages: Understanding C++ helps understand Java, C#, and other OOP languages
  • Career Opportunities: High demand in gaming, finance, systems programming, and IoT
  • Control: Fine-grained control over system resources and memory
  • Rich Ecosystem: Extensive libraries and frameworks available

Your First C++ Program

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Explanation:

  • #include <iostream> - Includes the input/output stream library
  • using namespace std; - Allows use of standard library names without std:: prefix
  • int main() - Main function where program execution begins
  • cout - Standard output stream object
  • << - Stream insertion operator
  • endl - Inserts newline and flushes the stream
  • return 0; - Indicates successful program execution

2. Variables & Data Types

What are Variables?

Variables are named storage locations in memory that hold data values. In C++, variables must be declared with a specific data type before they can be used. C++ is a strongly-typed language, meaning the type of a variable is checked at compile time.

Variable Declaration Syntax:

data_type variable_name;
data_type variable_name = initial_value;

Data Type Modifiers:

  • signed: Can store positive and negative values (default for int, char)
  • unsigned: Can store only positive values

  • long: Larger size (at least 4 bytes)
  • long long: Very large size (at least 8 bytes)

Example: Using Operators

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5;
    
    // Arithmetic operations
    cout << "Addition: " << a + b << endl;
    cout << "Subtraction: " << a - b << endl;
    cout << "Multiplication: " << a * b << endl;
    cout << "Division: " << a / b << endl;
    cout << "Modulus: " << a % b << endl;
    
    // Increment and decrement
    int x = 5;
    cout << "x++: " << x++ << endl;  // Post-increment
    cout << "x: " << x << endl;
    cout << "++x: " << ++x << endl;  // Pre-increment
    
    // Relational operations
    cout << "a > b: " << (a > b) << endl;
    cout << "a == b: " << (a == b) << endl;
    
    // Logical operations
    cout << "a > 5 && b < 10: " << (a > 5 && b < 10) << endl;
    
    // Assignment operators
    int c = 10;
    c += 5;  // c = c + 5
    cout << "c after += 5: " << c << endl;
    
    return 0;
}

4. Control Flow

If-Else Statement

The if-else statement allows you to execute different code blocks based on conditions.

Example: If-Else

#include <iostream>
using namespace std;

int main() {
    int number = 10;
    
    if (number > 0) {
        cout << "Number is positive" << endl;
    } else if (number < 0) {
        cout << "Number is negative" << endl;
    } else {
        cout << "Number is zero" << endl;
    }
    
    return 0;
}

Switch Statement

The switch statement allows you to select one of many code blocks to execute based on a value.

Example: Switch

#include <iostream>
using namespace std;

int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            cout << "Monday" << endl;
            break;
        case 2:
            cout << "Tuesday" << endl;
            break;
        case 3:
            cout << "Wednesday" << endl;
            break;
        default:
            cout << "Invalid day" << endl;
    }
    
    return 0;
}

Loops in C++

1. For Loop

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    // Output: 1 2 3 4 5
    return 0;
}

2. While Loop

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 5) {
        cout << i << " ";
        i++;
    }
    return 0;
}

3. Do-While Loop

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    do {
        cout << i << " ";
        i++;
    } while (i <= 5);
    return 0;
}

4. Range-based For Loop (C++11)

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> numbers = {1, 2, 3, 4, 5};
    
    for (int num : numbers) {
        cout << num << " ";
    }
    return 0;
}

Jump Statements

  • break: Exits the loop or switch statement
  • continue: Skips the rest of the loop and continues to the next iteration
  • return: Exits the function
  • goto: Jumps to a labeled statement (not recommended)

5. Functions

What are Functions?

Functions are blocks of code that perform a specific task. They help in code reusability, modularity, and organization. In C++, functions can be defined before or after the main function, but if defined after, they must be declared (prototyped) before use.

Function Syntax

return_type function_name(parameters) {
    // function body
    return value;
}

Example: Simple Function

#include <iostream>
using namespace std;

// Function declaration (prototype)
int add(int a, int b);

int main() {
    int result = add(5, 3);
    cout << "Sum: " << result << endl;
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Function Overloading

C++ allows multiple functions with the same name but different parameters (function overloading).

#include <iostream>
using namespace std;

// Overloaded functions
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int add(int a, int b, int c) {
    return a + b + c;
}

int main() {
    cout << add(5, 3) << endl;        // Calls int add(int, int)
    cout << add(5.5, 3.2) << endl;    // Calls double add(double, double)
    cout << add(1, 2, 3) << endl;     // Calls int add(int, int, int)
    return 0;
}

Default Parameters

C++ allows functions to have default parameter values.

#include <iostream>
using namespace std;

void greet(string name = "Guest") {
    cout << "Hello, " << name << "!" << endl;
}

int main() {
    greet();           // Uses default: "Hello, Guest!"
    greet("John");     // "Hello, John!"
    return 0;
}

Inline Functions

Inline functions suggest the compiler to replace the function call with the function code for better performance.

#include <iostream>
using namespace std;

inline int square(int x) {
    return x * x;
}

int main() {
    cout << square(5) << endl;
    return 0;
}

6. Object-Oriented Programming

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects" that contain data (attributes) and code (methods). C++ supports OOP through classes and objects.

Key OOP Concepts:

  • Class: Blueprint for creating objects
  • Object: Instance of a class
  • Encapsulation: Bundling data and methods together
  • Inheritance: Creating new classes from existing ones
  • Polymorphism: Same interface, different implementations
  • Abstraction: Hiding implementation details

Class and Object Example

#include <iostream>
#include <string>
using namespace std;

class Student {
private:
    string name;
    int age;
    
public:
    // Constructor
    Student(string n, int a) {
        name = n;
        age = a;
    }
    
    // Getter methods
    string getName() {
        return name;
    }
    
    int getAge() {
        return age;
    }
    
    // Setter methods
    void setName(string n) {
        name = n;
    }
    
    void setAge(int a) {
        age = a;
    }
    
    // Display method
    void display() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

int main() {
    // Creating objects
    Student s1("John", 20);
    Student s2("Alice", 22);
    
    s1.display();
    s2.display();
    
    return 0;
}

Inheritance Example

#include <iostream>
#include <string>
using namespace std;

// Base class
class Animal {
public:
    void eat() {
        cout << "Animal is eating" << endl;
    }
};

// Derived class
class Dog : public Animal {
public:
    void bark() {
        cout << "Dog is barking" << endl;
    }
};

int main() {
    Dog myDog;
    myDog.eat();   // Inherited from Animal
    myDog.bark();  // Own method
    return 0;
}

Access Specifiers

  • public: Accessible from anywhere
  • private: Accessible only within the class
  • protected: Accessible within the class and derived classes

7. Templates

What are Templates?

Templates are a powerful feature in C++ that allows you to write generic code. They enable you to define functions and classes that work with any data type without rewriting code for each type.

Function Templates

Function templates allow you to create generic functions that work with different data types.

#include <iostream>
using namespace std;

// Function template
template <typename T>
T maximum(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    cout << maximum(5, 10) << endl;           // int
    cout << maximum(5.5, 3.2) << endl;        // double
    cout << maximum('a', 'z') << endl;        // char
    return 0;
}

Class Templates

Class templates allow you to create generic classes.

#include <iostream>
using namespace std;

template <class T>
class Calculator {
private:
    T num1, num2;
    
public:
    Calculator(T a, T b) {
        num1 = a;
        num2 = b;
    }
    
    T add() {
        return num1 + num2;
    }
    
    T multiply() {
        return num1 * num2;
    }
};

int main() {
    Calculator<int> calc1(10, 5);
    Calculator<double> calc2(10.5, 5.5);
    
    cout << calc1.add() << endl;
    cout << calc2.multiply() << endl;
    return 0;
}

Template Specialization

Template specialization allows you to provide a specific implementation for a particular data type.

#include <iostream>
using namespace std;

template <typename T>
void print(T value) {
    cout << value << endl;
}

// Specialization for strings
template <>
void print<string>(string value) {
    cout << "String: " << value << endl;
}

int main() {
    print(5);              // Generic template
    print("Hello");        // Specialized template
    return 0;
}

8. STL (Standard Template Library)

What is STL?

The Standard Template Library (STL) is a powerful set of C++ template classes that provide common data structures and algorithms. STL consists of four main components: containers, iterators, algorithms, and function objects.

STL Components:

  • Containers: Data structures like vector, list, map, set
  • Iterators: Objects that point to elements in containers
  • Algorithms: Functions that operate on containers
  • Function Objects: Objects that can be called like functions

1. Vector (Dynamic Array)

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> numbers;
    
    // Adding elements
    numbers.push_back(10);
    numbers.push_back(20);
    numbers.push_back(30);
    
    // Accessing elements
    cout << numbers[0] << endl;
    cout << numbers.size() << endl;
    
    // Iterating
    for (int i = 0; i < numbers.size(); i++) {
        cout << numbers[i] << " ";
    }
    
    return 0;
}

2. List (Doubly Linked List)

#include <iostream>
#include <list>
using namespace std;

int main() {
    list<int> numbers = {10, 20, 30};
    
    numbers.push_back(40);
    numbers.push_front(5);
    
    for (int num : numbers) {
        cout << num << " ";
    }
    
    return 0;
}

3. Map (Key-Value Pairs)

#include <iostream>
#include <map>
#include <string>
using namespace std;

int main() {
    map<string, int> ages;
    
    ages["John"] = 25;
    ages["Alice"] = 30;
    ages["Bob"] = 22;
    
    cout << ages["John"] << endl;
    
    // Iterating
    for (auto pair : ages) {
        cout << pair.first << ": " << pair.second << endl;
    }
    
    return 0;
}

4. STL Algorithms

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> numbers = {5, 2, 8, 1, 9};
    
    // Sorting
    sort(numbers.begin(), numbers.end());
    
    // Finding
    auto it = find(numbers.begin(), numbers.end(), 8);
    
    // Counting
    int count = count_if(numbers.begin(), numbers.end(), 
                        [](int x) { return x > 5; });
    
    for (int num : numbers) {
        cout << num << " ";
    }
    
    return 0;
}
