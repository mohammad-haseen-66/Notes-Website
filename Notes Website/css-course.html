1. Introduction to CSS3

What is CSS3?

CSS3 is the latest version of Cascading Style Sheets, used to style and format web pages. It introduces many new features and improvements over CSS2.

Key Features:

  • Flexbox: Flexible box layout
  • Grid: Two-dimensional layout system
  • Animations: Keyframe animations
  • Transitions: Smooth property changes
  • Media Queries: Responsive design

Basic CSS Syntax

selector {
    property: value;
    property: value;
}

2. CSS Selectors

What are CSS Selectors?

CSS selectors are patterns used to select and style HTML elements. They define which elements on a page will be styled by the CSS rules. Understanding selectors is fundamental to writing effective CSS.

Basic Selectors

Element Selector

Selects all elements of a specific type:

/* Selects all <p> elements */
p {
    color: blue;
    font-size: 16px;
}

/* Selects all <h1> elements */
h1 {
    color: red;
    text-align: center;
}

ID Selector

Selects an element with a specific ID (unique identifier). Uses the hash symbol (#):

/* Selects element with id="header" */
#header {
    background-color: #333;
    color: white;
    padding: 20px;
}

/* HTML: <div id="header">Header</div> */

Class Selector

Selects elements with a specific class. Uses the dot symbol (.):

/* Selects all elements with class="button" */
.button {
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    border: none;
    cursor: pointer;
}

/* HTML: <button class="button">Click Me</button> */

Universal Selector

Selects all elements on the page. Uses the asterisk (*):

/* Selects all elements */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Combinator Selectors

Descendant Selector (Space)

Selects all elements that are descendants of a specified element:

/* Selects all <p> elements inside <div> */
div p {
    color: blue;
}

/* HTML: <div><p>This is blue</p></div> */

Child Selector (>)

Selects direct children of a specified element:

/* Selects only direct <li> children of <ul> */
ul > li {
    list-style-type: square;
}

/* HTML: <ul><li>Item</li></ul> */

Adjacent Sibling Selector (+)

Selects an element that immediately follows another element:

/* Selects <p> that immediately follows <h2> */
h2 + p {
    margin-top: 10px;
    font-weight: bold;
}

General Sibling Selector (~)

Selects all siblings that follow a specified element:

/* Selects all <p> elements that are siblings of <h2> */
h2 ~ p {
    color: gray;
}

Attribute Selectors

Select elements based on their attributes:

/* Selects elements with specific attribute */
input[type="text"] {
    border: 1px solid #ccc;
}

/* Selects elements with attribute present */
a[target] {
    color: red;
}

/* Selects elements with attribute containing value */
a[href*="example"] {
    font-weight: bold;
}

/* Selects elements with attribute starting with value */
a[href^="https"] {
    color: green;
}

/* Selects elements with attribute ending with value */
img[src$=".jpg"] {
    border: 2px solid blue;
}

Pseudo-class Selectors

Select elements in a specific state:

/* Link states */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: orange; }

/* Form states */
input:focus {
    outline: 2px solid blue;
    border-color: blue;
}

input:disabled {
    opacity: 0.5;
    cursor: not-allowed;
}

/* Structural pseudo-classes */
p:first-child { font-weight: bold; }
p:last-child { margin-bottom: 0; }
p:nth-child(2) { color: red; }
p:nth-child(even) { background: #f0f0f0; }
p:nth-child(odd) { background: white; }

Pseudo-element Selectors

Style specific parts of an element:

/* First line of paragraph */
p::first-line {
    font-weight: bold;
    color: blue;
}

/* First letter of paragraph */
p::first-letter {
    font-size: 2em;
    color: red;
}

/* Before and after content */
h1::before {
    content: "‚Üí ";
    color: green;
}

h1::after {
    content: " ‚Üê";
    color: green;
}

/* Selection styling */
::selection {
    background-color: yellow;
    color: black;
}

Selector Specificity

When multiple rules apply to the same element, CSS uses specificity to determine which rule takes precedence:

  • Inline styles: 1000 points
  • ID selectors: 100 points
  • Class, attribute, pseudo-class: 10 points
  • Element, pseudo-element: 1 point

3. CSS Properties

What are CSS Properties?

CSS properties define the visual appearance and behavior of HTML elements. Each property has a name and a value, and together they create the styling rules for web pages.

Text Properties

Color

Sets the text color:

/* Named colors */
p { color: red; }

/* Hexadecimal */
h1 { color: #FF5733; }

/* RGB */
h2 { color: rgb(255, 87, 51); }

/* RGBA (with transparency) */
h3 { color: rgba(255, 87, 51, 0.5); }

Font Properties

/* Font family */
body {
    font-family: Arial, Helvetica, sans-serif;
}

/* Font size */
h1 { font-size: 2em; }
h2 { font-size: 1.5em; }
p { font-size: 16px; }

/* Font weight */
.bold { font-weight: bold; }
.light { font-weight: 300; }

/* Font style */
.italic { font-style: italic; }

/* Text alignment */
.center { text-align: center; }
.left { text-align: left; }
.right { text-align: right; }
.justify { text-align: justify; }

/* Text decoration */
.underline { text-decoration: underline; }
.line-through { text-decoration: line-through; }
.no-decoration { text-decoration: none; }

/* Line height */
p {
    line-height: 1.6;
    letter-spacing: 1px;
    word-spacing: 2px;
}

Box Model Properties

The CSS box model consists of content, padding, border, and margin:

Width and Height

.box {
    width: 300px;
    height: 200px;
    min-width: 200px;
    max-width: 500px;
    min-height: 100px;
    max-height: 400px;
}

Padding

Space between content and border:

/* Individual sides */
.element {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
}

/* Shorthand */
.element {
    padding: 10px;              /* all sides */
    padding: 10px 20px;         /* top/bottom left/right */
    padding: 10px 20px 15px;    /* top left/right bottom */
    padding: 10px 20px 15px 25px; /* top right bottom left */
}

Margin

Space outside the border:

/* Individual sides */
.element {
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;
}

/* Shorthand (same as padding) */
.element {
    margin: 10px;
    margin: 10px 20px;
    margin: 10px 20px 15px;
    margin: 10px 20px 15px 25px;
}

/* Auto for centering */
.center {
    margin: 0 auto;
    width: 500px;
}

Border

/* Individual properties */
.element {
    border-width: 2px;
    border-style: solid;
    border-color: #333;
}

/* Shorthand */
.element {
    border: 2px solid #333;
}

/* Individual sides */
.element {
    border-top: 2px solid red;
    border-right: 3px dashed blue;
    border-bottom: 1px dotted green;
    border-left: 4px double orange;
}

/* Border radius (rounded corners) */
.rounded {
    border-radius: 10px;
}
.circle {
    border-radius: 50%;
}

Box-sizing

/* Default: content-box (width includes only content) */
.element {
    box-sizing: content-box;
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Total width = 300 + 40 + 10 = 350px */
}

/* border-box (width includes padding and border) */
.element {
    box-sizing: border-box;
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Total width = 300px (includes padding and border) */
}

Background Properties

/* Background color */
body {
    background-color: #f0f0f0;
}

/* Background image */
.header {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}

/* Shorthand */
.header {
    background: #f0f0f0 url('image.jpg') no-repeat center/cover;
}

/* Multiple backgrounds */
.element {
    background: 
        linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
        url('image.jpg');
}

Display Property

/* Block: takes full width, starts on new line */
.block {
    display: block;
}

/* Inline: only takes needed width, stays in line */
.inline {
    display: inline;
}

/* Inline-block: inline but can have width/height */
.inline-block {
    display: inline-block;
    width: 200px;
    height: 100px;
}

/* None: element is hidden */
.hidden {
    display: none;
}

/* Flex: enables flexbox layout */
.flex-container {
    display: flex;
}

/* Grid: enables grid layout */
.grid-container {
    display: grid;
}

Position Property

/* Static: default, normal flow */
.static {
    position: static;
}

/* Relative: relative to normal position */
.relative {
    position: relative;
    top: 20px;
    left: 30px;
}

/* Absolute: relative to nearest positioned parent */
.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fixed: relative to viewport */
.fixed {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
}

/* Sticky: hybrid of relative and fixed */
.sticky {
    position: sticky;
    top: 0;
}

Z-index

Controls stacking order of positioned elements:

.overlay {
    position: absolute;
    z-index: 1000;
}

.modal {
    position: fixed;
    z-index: 2000;
}

4. Layout (Flexbox & Grid)

CSS Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns. It provides a flexible way to distribute space and align items.

Flex Container Properties

.flex-container {
    display: flex;
    
    /* Direction */
    flex-direction: row;        /* default: left to right */
    flex-direction: row-reverse; /* right to left */
    flex-direction: column;     /* top to bottom */
    flex-direction: column-reverse; /* bottom to top */
    
    /* Wrapping */
    flex-wrap: nowrap;          /* default: no wrap */
    flex-wrap: wrap;            /* wrap to next line */
    flex-wrap: wrap-reverse;    /* wrap in reverse */
    
    /* Shorthand for direction and wrap */
    flex-flow: row wrap;
    
    /* Justify content (main axis) */
    justify-content: flex-start;    /* default: start */
    justify-content: flex-end;      /* end */
    justify-content: center;        /* center */
    justify-content: space-between;  /* space between items */
    justify-content: space-around;   /* space around items */
    justify-content: space-evenly;   /* equal space */
    
    /* Align items (cross axis) */
    align-items: stretch;       /* default: stretch */
    align-items: flex-start;   /* start */
    align-items: flex-end;     /* end */
    align-items: center;       /* center */
    align-items: baseline;     /* baseline */
    
    /* Align content (for multiple lines) */
    align-content: flex-start;
    align-content: flex-end;
    align-content: center;
    align-content: space-between;
    align-content: space-around;
    align-content: stretch;
}

Flex Item Properties

.flex-item {
    /* Grow factor */
    flex-grow: 0;        /* default: don't grow */
    flex-grow: 1;        /* grow to fill space */
    
    /* Shrink factor */
    flex-shrink: 1;      /* default: can shrink */
    flex-shrink: 0;      /* don't shrink */
    
    /* Basis */
    flex-basis: auto;    /* default: auto */
    flex-basis: 200px;   /* initial size */
    
    /* Shorthand */
    flex: 1;             /* grow: 1, shrink: 1, basis: 0% */
    flex: 0 1 200px;     /* grow: 0, shrink: 1, basis: 200px */
    
    /* Align self */
    align-self: auto;
    align-self: flex-start;
    align-self: flex-end;
    align-self: center;
    align-self: stretch;
    
    /* Order */
    order: 0;            /* default: 0 */
    order: 1;            /* display after items with order 0 */
}

Flexbox Example

.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 20px;
}

.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    justify-content: center;
}

.card {
    flex: 1 1 300px;  /* grow, shrink, basis */
    max-width: 400px;
}

CSS Grid

Grid is a two-dimensional layout system that allows you to create complex layouts with rows and columns.

Grid Container Properties

.grid-container {
    display: grid;
    
    /* Define columns */
    grid-template-columns: 200px 200px 200px;
    grid-template-columns: 1fr 1fr 1fr;        /* equal fractions */
    grid-template-columns: repeat(3, 1fr);     /* repeat 3 times */
    grid-template-columns: 1fr 2fr 1fr;        /* different sizes */
    grid-template-columns: auto auto auto;     /* auto size */
    
    /* Define rows */
    grid-template-rows: 100px 200px 100px;
    grid-template-rows: repeat(3, 1fr);
    
    /* Shorthand */
    grid-template: 
        "header header header" 100px
        "sidebar main aside" 1fr
        "footer footer footer" 50px / 200px 1fr 200px;
    
    /* Gap between items */
    gap: 20px;              /* row and column gap */
    row-gap: 20px;          /* row gap only */
    column-gap: 20px;       /* column gap only */
    
    /* Alignment */
    justify-items: start;    /* align items horizontally */
    justify-items: end;
    justify-items: center;
    justify-items: stretch;
    
    align-items: start;     /* align items vertically */
    align-items: end;
    align-items: center;
    align-items: stretch;
    
    /* Content alignment */
    justify-content: start;
    justify-content: end;
    justify-content: center;
    justify-content: space-between;
    
    align-content: start;
    align-content: end;
    align-content: center;
    align-content: space-between;
}

Grid Item Properties

.grid-item {
    /* Position using line numbers */
    grid-column-start: 1;
    grid-column-end: 3;
    grid-row-start: 1;
    grid-row-end: 2;
    
    /* Shorthand */
    grid-column: 1 / 3;      /* start / end */
    grid-row: 1 / 2;
    
    /* Span */
    grid-column: 1 / span 2;  /* start at 1, span 2 columns */
    grid-row: 2 / span 3;     /* start at 2, span 3 rows */
    
    /* Using named areas */
    grid-area: header;
    
    /* Alignment */
    justify-self: start;
    justify-self: end;
    justify-self: center;
    justify-self: stretch;
    
    align-self: start;
    align-self: end;
    align-self: center;
    align-self: stretch;
}

Grid Example

.page-layout {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header header"
        "sidebar main aside"
        "footer footer footer";
    gap: 20px;
    min-height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

Flexbox vs Grid

Feature Flexbox Grid
Dimension One-dimensional (row or column) Two-dimensional (rows and columns)
Use Case Component layout, navigation Page layout, complex structures
Alignment Good for single axis Excellent for both axes
Complexity Simpler for basic layouts Better for complex layouts

5. Responsive Design

What is Responsive Design?

Responsive design ensures that web pages look good and function well on all devices and screen sizes. It uses flexible layouts, flexible images, and CSS media queries to adapt to different viewport sizes.

Viewport Meta Tag

Essential for responsive design - must be included in HTML head:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Media queries allow you to apply CSS rules based on device characteristics like screen width, height, orientation, etc.

Basic Media Query Syntax

/* Apply styles when screen width is at least 768px */
@media (min-width: 768px) {
    .container {
        max-width: 1200px;
        margin: 0 auto;
    }
}

/* Apply styles when screen width is at most 767px */
@media (max-width: 767px) {
    .container {
        padding: 10px;
    }
}

/* Range of screen sizes */
@media (min-width: 768px) and (max-width: 1024px) {
    .container {
        max-width: 900px;
    }
}

/* Orientation */
@media (orientation: landscape) {
    .sidebar {
        width: 300px;
    }
}

@media (orientation: portrait) {
    .sidebar {
        width: 100%;
    }
}

Common Breakpoints

/* Mobile First Approach */
/* Base styles for mobile */
.container {
    width: 100%;
    padding: 10px;
}

/* Small devices (tablets, 768px and up) */
@media (min-width: 768px) {
    .container {
        max-width: 750px;
        padding: 20px;
    }
}

/* Medium devices (desktops, 992px and up) */
@media (min-width: 992px) {
    .container {
        max-width: 970px;
    }
}

/* Large devices (large desktops, 1200px and up) */
@media (min-width: 1200px) {
    .container {
        max-width: 1170px;
    }
}

Flexible Units

Using relative units instead of fixed pixels:

/* Percentage - relative to parent */
.container {
    width: 100%;
    max-width: 1200px;
}

/* Viewport units */
.header {
    height: 100vh;        /* 100% of viewport height */
    width: 100vw;        /* 100% of viewport width */
}

.title {
    font-size: 5vw;     /* 5% of viewport width */
}

/* Relative units */
.text {
    font-size: 1rem;    /* relative to root */
    padding: 1em;       /* relative to parent font-size */
}

/* Flexible images */
img {
    max-width: 100%;
    height: auto;
}

Responsive Typography

/* Fluid typography using clamp */
h1 {
    font-size: clamp(2rem, 5vw, 4rem);
    /* min: 2rem, preferred: 5vw, max: 4rem */
}

/* Responsive with media queries */
body {
    font-size: 16px;
}

@media (min-width: 768px) {
    body {
        font-size: 18px;
    }
}

@media (min-width: 1200px) {
    body {
        font-size: 20px;
    }
}

Responsive Grid and Flexbox

/* Mobile: single column */
.grid {
    display: grid;
    grid-template-columns: 1fr;
    gap: 20px;
}

/* Tablet: 2 columns */
@media (min-width: 768px) {
    .grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Desktop: 3 columns */
@media (min-width: 1024px) {
    .grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

/* Flexbox responsive */
.flex-container {
    display: flex;
    flex-direction: column;
}

@media (min-width: 768px) {
    .flex-container {
        flex-direction: row;
    }
}

Responsive Navigation

/* Mobile: hamburger menu */
.nav {
    display: none;
}

.menu-toggle {
    display: block;
}

/* Desktop: horizontal menu */
@media (min-width: 768px) {
    .nav {
        display: flex;
    }
    
    .menu-toggle {
        display: none;
    }
}

Mobile-First vs Desktop-First

Approach Method Media Query
Mobile-First Start with mobile styles, add desktop styles min-width
Desktop-First Start with desktop styles, override for mobile max-width

6. Animations

CSS Transitions

Transitions allow you to change property values smoothly over a specified duration.

Transition Properties

.element {
    /* Property to transition */
    transition-property: background-color;
    
    /* Duration */
    transition-duration: 0.3s;
    
    /* Timing function */
    transition-timing-function: ease;
    
    /* Delay */
    transition-delay: 0.1s;
    
    /* Shorthand */
    transition: background-color 0.3s ease 0.1s;
    
    /* Multiple properties */
    transition: 
        background-color 0.3s ease,
        transform 0.5s ease-in-out,
        opacity 0.2s linear;
    
    /* All properties */
    transition: all 0.3s ease;
}

/* Trigger on hover */
.button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: red;
}

Timing Functions

.element {
    /* Ease (default) - slow start, fast middle, slow end */
    transition-timing-function: ease;
    
    /* Linear - constant speed */
    transition-timing-function: linear;
    
    /* Ease-in - slow start */
    transition-timing-function: ease-in;
    
    /* Ease-out - slow end */
    transition-timing-function: ease-out;
    
    /* Ease-in-out - slow start and end */
    transition-timing-function: ease-in-out;
    
    /* Cubic bezier - custom curve */
    transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1);
    
    /* Steps - discrete steps */
    transition-timing-function: steps(4);
}

CSS Animations

Animations are more powerful than transitions and allow you to create complex animations with keyframes.

Keyframes

/* Define animation */
@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

/* Using percentages */
@keyframes bounce {
    0% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-50px);
    }
    100% {
        transform: translateY(0);
    }
}

/* Multiple keyframes */
@keyframes colorChange {
    0% { background-color: red; }
    33% { background-color: yellow; }
    66% { background-color: blue; }
    100% { background-color: green; }
}

Animation Properties

.animated-element {
    /* Animation name */
    animation-name: slideIn;
    
    /* Duration */
    animation-duration: 1s;
    
    /* Timing function */
    animation-timing-function: ease;
    
    /* Delay */
    animation-delay: 0.5s;
    
    /* Iteration count */
    animation-iteration-count: 3;      /* number of times */
    animation-iteration-count: infinite; /* forever */
    
    /* Direction */
    animation-direction: normal;       /* forward */
    animation-direction: reverse;       /* backward */
    animation-direction: alternate;     /* forward then backward */
    animation-direction: alternate-reverse; /* backward then forward */
    
    /* Fill mode */
    animation-fill-mode: none;           /* default */
    animation-fill-mode: forwards;       /* keep end state */
    animation-fill-mode: backwards;     /* keep start state */
    animation-fill-mode: both;          /* keep both states */
    
    /* Play state */
    animation-play-state: running;      /* playing */
    animation-play-state: paused;       /* paused */
    
    /* Shorthand */
    animation: slideIn 1s ease 0.5s 3 alternate forwards;
    /* name duration timing-function delay iteration-count direction fill-mode */
}

Transform Property

Transform allows you to rotate, scale, skew, or translate elements:

.element {
    /* Translate (move) */
    transform: translateX(50px);      /* move right */
    transform: translateY(50px);      /* move down */
    transform: translate(50px, 50px); /* move both */
    
    /* Rotate */
    transform: rotate(45deg);         /* rotate 45 degrees */
    transform: rotateZ(90deg);        /* rotate on Z axis */
    
    /* Scale */
    transform: scale(1.5);            /* 1.5x larger */
    transform: scaleX(2);             /* 2x width */
    transform: scaleY(0.5);           /* 0.5x height */
    transform: scale(2, 0.5);         /* width 2x, height 0.5x */
    
    /* Skew */
    transform: skewX(20deg);         /* skew horizontally */
    transform: skewY(10deg);         /* skew vertically */
    transform: skew(20deg, 10deg);   /* skew both */
    
    /* Multiple transforms */
    transform: translate(50px, 50px) rotate(45deg) scale(1.2);
    
    /* 3D transforms */
    transform: perspective(1000px) rotateY(45deg);
    transform-style: preserve-3d;
}
