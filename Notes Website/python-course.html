1. Introduction to Python

What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991. Python emphasizes code readability with its notable use of significant whitespace and simple syntax that allows programmers to express concepts in fewer lines of code.

Python is an interpreted language, meaning that Python code is executed line by line at runtime, making it excellent for rapid development and prototyping. It supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Key Characteristics of Python:

  • Easy to Learn: Python has a simple and readable syntax that is easy for beginners
  • Interpreted Language: Code is executed directly without compilation
  • Dynamically Typed: Variable types are determined at runtime
  • Versatile: Used for web development, data science, AI, automation, and more
  • Large Standard Library: Comes with extensive built-in modules
  • Cross-platform: Runs on Windows, Linux, macOS, and other platforms

Why Learn Python?

  • Popularity: One of the most popular programming languages worldwide
  • Career Opportunities: High demand in data science, AI, web development, and automation
  • Easy Syntax: Simple and readable code makes it perfect for beginners
  • Versatile Applications: Used in web development, data analysis, machine learning, automation, and more
  • Strong Community: Large community and extensive documentation
  • Rapid Development: Faster development cycle compared to compiled languages

Your First Python Program

print("Hello, World!")

Explanation:

  • print() - Built-in function to display output to the console
  • "Hello, World!" - String literal (text enclosed in quotes)
  • No semicolons or braces needed - Python uses indentation for code blocks

2. Variables & Data Types

What are Variables?

Variables are containers that store data values. In Python, you don't need to declare the type of a variable. Python automatically determines the type based on the value assigned to it.

Variable Naming Rules:

  • Variable names can contain letters, numbers, and underscores
  • Must start with a letter or underscore (not a number)
  • Case-sensitive (age and Age are different)
  • Cannot use Python keywords as variable names
  • Use descriptive names (e.g., user_name instead of un)

Basic Data Types in Python

str String (text) name = "John"
bool Boolean (True/False) is_student = True
list Ordered collection of items numbers = [1, 2, 3]
tuple Immutable ordered collection coordinates = (10, 20)
dict Key-value pairs person = {"name": "John"}

3. Operators

What are Operators?

Operators are special symbols that perform operations on variables and values. Python supports various types of operators for different operations.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations:

Operator Description Example
+ Addition 5 + 3 = 8
- Subtraction 5 - 3 = 2
* Multiplication 5 * 3 = 15
/ Division 5 / 3 = 1.666...
% Modulus (remainder) 5 % 3 = 2
** Exponentiation 5 ** 3 = 125
// Floor division 5 // 3 = 1

Comparison Operators

Comparison operators compare two values and return a boolean result:

Operator Description Example
== Equal to 5 == 5 â†’ True
!= Not equal to 5 != 3 â†’ True
< Less than 3 < 5 â†’ True
> Greater than 5 > 3 â†’ True
<= Less than or equal to 5 <= 5 â†’ True
>= Greater than or equal to 5 >= 3 â†’ True

Logical Operators

Logical operators are used to combine conditional statements:

Operator Description Example
and Returns True if both statements are true True and False â†’ False
or Returns True if one of the statements is true True or False â†’ True
not Reverse the result not True â†’ False

Assignment Operators

Assignment operators are used to assign values to variables:

# Basic assignment
      x = 5
      
      # Addition assignment
      x += 3  # Same as x = x + 3
      
      # Subtraction assignment
      x -= 2  # Same as x = x - 2
      
      # Multiplication assignment
      x *= 2  # Same as x = x * 2
      
      # Division assignment
      x /= 2  # Same as x = x / 2
      
      # Modulus assignment
      x %= 3  # Same as x = x % 3
      
      # Exponentiation assignment
      x **= 2  # Same as x = x ** 2
      
      # Floor division assignment
      x //= 2  # Same as x = x // 2

Identity Operators

Identity operators compare the memory locations of two objects:

# is - Returns True if both variables are the same object
      x = [1, 2, 3]
      y = x
      print(x is y)  # True
      
      # is not - Returns True if both variables are not the same object
      a = [1, 2, 3]
      b = [1, 2, 3]
      print(a is not b)  # True (different objects)

Membership Operators

Membership operators test if a value is present in a sequence:

# in - Returns True if value is found in sequence
      fruits = ["apple", "banana", "cherry"]
      print("banana" in fruits)  # True
      
      # not in - Returns True if value is not found in sequence
      print("orange" not in fruits)  # True

4. Control Flow

What is Control Flow?

Control flow statements allow you to control the order in which your code executes. Python provides several control flow statements including if-else, loops, and break/continue.

If-Else Statements

The if statement is used to execute code conditionally:

# Simple if statement
      age = 18
      if age >= 18:
          print("You are an adult")
      
      # If-else statement
      age = 16
      if age >= 18:
          print("You are an adult")
      else:
          print("You are a minor")
      
      # If-elif-else statement
      score = 85
      if score >= 90:
          print("Grade: A")
      elif score >= 80:
          print("Grade: B")
      elif score >= 70:
          print("Grade: C")
      else:
          print("Grade: F")

For Loops

For loops are used to iterate over a sequence (list, tuple, string, etc.):

# Iterating over a list
      fruits = ["apple", "banana", "cherry"]
      for fruit in fruits:
          print(fruit)
      
      # Using range() function
      for i in range(5):
          print(i)  # Prints 0, 1, 2, 3, 4
      
      # Range with start and end
      for i in range(2, 6):
          print(i)  # Prints 2, 3, 4, 5
      
      # Range with step
      for i in range(0, 10, 2):
          print(i)  # Prints 0, 2, 4, 6, 8
      
      # Iterating with index
      fruits = ["apple", "banana", "cherry"]
      for index, fruit in enumerate(fruits):
          print(f"{index}: {fruit}")

While Loops

While loops execute as long as a condition is true:

# Basic while loop
      count = 0
      while count < 5:
          print(count)
          count += 1
      
      # While loop with break
      count = 0
      while True:
          print(count)
          count += 1
          if count >= 5:
              break
      
      # While loop with continue
      count = 0
      while count < 5:
          count += 1
          if count == 3:
              continue
          print(count)

Break and Continue

break exits the loop, while continue skips the current iteration:

# Using break
      for i in range(10):
          if i == 5:
              break
          print(i)  # Prints 0, 1, 2, 3, 4
      
      # Using continue
      for i in range(10):
          if i == 5:
              continue
          print(i)  # Prints 0, 1, 2, 3, 4, 6, 7, 8, 9

Nested Loops

You can nest loops inside other loops:

# Nested for loops
      for i in range(3):
          for j in range(3):
              print(f"({i}, {j})")
      
      # Creating a multiplication table
      for i in range(1, 6):
          for j in range(1, 6):
              print(f"{i * j:3}", end=" ")
          print()

Pass Statement

The pass statement is a null operation - it does nothing. It's used as a placeholder:

# Using pass as placeholder
      if age >= 18:
          pass  # Will implement later
      else:
          print("Minor")
      
      # Pass in function definition
      def my_function():
          pass  # Function body to be implemented later

5. Functions

What are Functions?

Functions are reusable blocks of code that perform a specific task. They help organize code, make it more readable, and avoid repetition.

Defining Functions

Functions are defined using the def keyword:

# Simple function
      def greet():
          print("Hello, World!")
      
      # Calling a function
      greet()  # Output: Hello, World!
      
      # Function with parameters
      def greet(name):
          print(f"Hello, {name}!")
      
      greet("Alice")  # Output: Hello, Alice!
      
      # Function with multiple parameters
      def add(a, b):
          return a + b
      
      result = add(5, 3)
      print(result)  # Output: 8

Return Statement

The return statement sends a value back to the caller:

# Function with return
      def multiply(x, y):
          return x * y
      
      result = multiply(4, 5)
      print(result)  # Output: 20
      
      # Function returning multiple values
      def get_name_age():
          return "John", 25
      
      name, age = get_name_age()
      print(f"{name} is {age} years old")

Default Parameters

You can provide default values for function parameters:

# Function with default parameter
      def greet(name, greeting="Hello"):
          print(f"{greeting}, {name}!")
      
      greet("Alice")              # Output: Hello, Alice!
      greet("Bob", "Hi")          # Output: Hi, Bob!
      
      # Multiple default parameters
      def create_profile(name, age=18, city="Unknown"):
          print(f"Name: {name}, Age: {age}, City: {city}")
      
      create_profile("Alice")                    # Uses defaults
      create_profile("Bob", 25)                  # Overrides age
      create_profile("Charlie", 30, "New York")  # Overrides all

Keyword Arguments

You can pass arguments by name using keyword arguments:

def introduce(name, age, city):
          print(f"{name} is {age} years old and lives in {city}")
      
      # Using keyword arguments
      introduce(name="Alice", age=25, city="London")
      introduce(city="Paris", name="Bob", age=30)  # Order doesn't matter

Arbitrary Arguments

You can accept any number of arguments using *args and **kwargs:

# *args for positional arguments
      def sum_all(*args):
          total = 0
          for num in args:
              total += num
          return total
      
      print(sum_all(1, 2, 3, 4, 5))  # Output: 15
      
      # **kwargs for keyword arguments
      def print_info(**kwargs):
          for key, value in kwargs.items():
              print(f"{key}: {value}")
      
      print_info(name="Alice", age=25, city="London")

Lambda Functions

Lambda functions are small anonymous functions defined with the lambda keyword:

# Lambda function
      square = lambda x: x ** 2
      print(square(5))  # Output: 25
      
      # Lambda with multiple parameters
      add = lambda x, y: x + y
      print(add(3, 4))  # Output: 7
      
      # Lambda in higher-order functions
      numbers = [1, 2, 3, 4, 5]
      squared = list(map(lambda x: x ** 2, numbers))
      print(squared)  # Output: [1, 4, 9, 16, 25]

Scope and Global Variables

Variables have different scopes - local and global:

# Global variable
      x = 10
      
      def my_function():
          # Local variable
          y = 5
          print(f"Local y: {y}")
          print(f"Global x: {x}")
      
      my_function()
      
      # Modifying global variable
      count = 0
      
      def increment():
          global count
          count += 1
      
      increment()
      print(count)  # Output: 1

6. Lists & Tuples

What are Lists?

Lists are ordered, mutable (changeable) collections of items. They can contain items of different data types.

Creating Lists

# Empty list
      my_list = []
      
      # List with items
      fruits = ["apple", "banana", "cherry"]
      numbers = [1, 2, 3, 4, 5]
      
      # List with mixed types
      mixed = [1, "hello", 3.14, True]
      
      # Using list() constructor
      numbers = list((1, 2, 3))

Accessing List Items

fruits = ["apple", "banana", "cherry"]
      
      # Access by index
      print(fruits[0])   # apple
      print(fruits[1])   # banana
      print(fruits[-1]) # cherry (negative indexing)
      
      # Slicing
      print(fruits[1:3])      # ['banana', 'cherry']
      print(fruits[:2])       # ['apple', 'banana']
      print(fruits[1:])      # ['banana', 'cherry']

Modifying Lists

fruits = ["apple", "banana", "cherry"]
      
      # Change item
      fruits[1] = "orange"
      
      # Add item
      fruits.append("grape")        # Add to end
      fruits.insert(1, "mango")     # Insert at index
      
      # Remove item
      fruits.remove("banana")       # Remove by value
      fruits.pop(1)                 # Remove by index
      fruits.pop()                  # Remove last item
      del fruits[0]                 # Delete by index
      
      # Extend list
      fruits.extend(["kiwi", "pear"])

List Methods

Method Description
append() Adds an element at the end
insert() Inserts element at specified position
remove() Removes first occurrence of value
pop() Removes element at specified position
clear() Removes all elements
index() Returns index of first occurrence
count() Returns count of element
sort() Sorts the list
reverse() Reverses the list
copy() Returns a copy of the list

List Comprehension

List comprehension provides a concise way to create lists:

# Traditional way
      squares = []
      for x in range(5):
          squares.append(x ** 2)
      
      # List comprehension
      squares = [x ** 2 for x in range(5)]
      
      # With condition
      evens = [x for x in range(10) if x % 2 == 0]
      
      # Nested list comprehension
      matrix = [[i * j for j in range(3)] for i in range(3)]

What are Tuples?

Tuples are ordered, immutable (unchangeable) collections of items. Once created, they cannot be modified.

Creating Tuples

# Empty tuple
      my_tuple = ()
      
      # Tuple with items
      fruits = ("apple", "banana", "cherry")
      numbers = (1, 2, 3, 4, 5)
      
      # Single item tuple (note the comma)
      single = (5,)
      
      # Using tuple() constructor
      numbers = tuple([1, 2, 3])

Accessing Tuple Items

fruits = ("apple", "banana", "cherry")
      
      # Access by index
      print(fruits[0])   # apple
      print(fruits[-1])  # cherry
      
      # Slicing
      print(fruits[1:3])  # ('banana', 'cherry')
      
      # Unpacking
      a, b, c = fruits
      print(a, b, c)  # apple banana cherry

Tuple Methods

fruits = ("apple", "banana", "cherry", "apple")
      
      # count() - Count occurrences
      print(fruits.count("apple"))  # 2
      
      # index() - Find index
      print(fruits.index("banana"))  # 1

Lists vs Tuples

Feature List Tuple
Mutability Mutable (changeable) Immutable (unchangeable)
Syntax Square brackets [] Parentheses ()
Performance Slower Faster
Memory More memory Less memory
Use Case When data needs to change When data should not change

7. Dictionaries

What are Dictionaries?

Dictionaries are unordered, mutable collections of key-value pairs. Each key is unique and maps to a value. Dictionaries are also known as associative arrays or hash maps.

Creating Dictionaries

# Empty dictionary
      my_dict = {}
      
      # Dictionary with items
      person = {
          "name": "John",
          "age": 30,
          "city": "New York"
      }
      
      # Using dict() constructor
      person = dict(name="John", age=30, city="New York")
      
      # Dictionary with different data types
      mixed = {
          "string": "hello",
          "number": 42,
          "list": [1, 2, 3],
          "nested": {"key": "value"}
      }

Accessing Dictionary Items

person = {"name": "John", "age": 30, "city": "New York"}
      
      # Access by key
      print(person["name"])        # John
      print(person.get("age"))     # 30
      
      # Using get() with default
      print(person.get("email", "Not found"))  # Not found
      
      # Get all keys
      print(person.keys())         # dict_keys(['name', 'age', 'city'])
      
      # Get all values
      print(person.values())       # dict_values(['John', 30, 'New York'])
      
      # Get all items
      print(person.items())        # dict_items([('name', 'John'), ...])

Modifying Dictionaries

person = {"name": "John", "age": 30}
      
      # Add or update item
      person["city"] = "New York"
      person["age"] = 31
      
      # Update multiple items
      person.update({"age": 32, "email": "john@example.com"})
      
      # Remove item
      del person["city"]
      person.pop("age")
      person.popitem()  # Removes last item
      
      # Clear dictionary
      person.clear()

Dictionary Methods

Method Description
get() Returns value for key, or default if not found
keys() Returns all keys
values() Returns all values
items() Returns all key-value pairs
update() Updates dictionary with another dictionary
pop() Removes and returns value for key
popitem() Removes and returns last key-value pair
clear() Removes all items
copy() Returns a copy of the dictionary

Iterating Over Dictionaries

person = {"name": "John", "age": 30, "city": "New York"}
      
      # Iterate over keys
      for key in person:
          print(key)
      
      # Iterate over keys explicitly
      for key in person.keys():
          print(key)
      
      # Iterate over values
      for value in person.values():
          print(value)
      
      # Iterate over key-value pairs
      for key, value in person.items():
          print(f"{key}: {value}")

Dictionary Comprehension

# Create dictionary from list
      squares = {x: x**2 for x in range(5)}
      # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
      
      # With condition
      evens = {x: x*2 for x in range(10) if x % 2 == 0}
      
      # From two lists
      keys = ['a', 'b', 'c']
      values = [1, 2, 3]
      my_dict = {k: v for k, v in zip(keys, values)}

Nested Dictionaries

# Nested dictionary
      students = {
          "student1": {
              "name": "Alice",
              "age": 20,
              "grades": [85, 90, 88]
          },
          "student2": {
              "name": "Bob",
              "age": 21,
              "grades": [92, 87, 95]
          }
      }
      
      # Accessing nested values
      print(students["student1"]["name"])
      print(students["student1"]["grades"][0])

8. Strings

What are Strings?

Strings are sequences of characters enclosed in quotes. In Python, strings are immutable, meaning they cannot be changed after creation.

Creating Strings

# Single quotes
      name = 'John'
      
      # Double quotes
      name = "John"
      
      # Triple quotes for multiline
      text = """This is a
      multiline string"""
      
      # Using str() constructor
      number = str(123)

String Indexing and Slicing

text = "Hello, World!"
      
      # Access by index
      print(text[0])    # H
      print(text[-1])   # ! (negative indexing)
      
      # Slicing
      print(text[0:5])   # Hello
      print(text[:5])    # Hello
      print(text[7:])    # World!
      print(text[::2])   # Hlo ol! (step by 2)

String Methods

Method Description
upper() Converts to uppercase
lower() Converts to lowercase
strip() Removes whitespace from ends
split() Splits string into list
join() Joins list into string
replace() Replaces substring
find() Finds index of substring
count() Counts occurrences
startswith() Checks if starts with substring
endswith() Checks if ends with substring

String Operations

# Concatenation
      first = "Hello"
      last = "World"
      greeting = first + ", " + last
      
      # Repetition
      text = "Hi" * 3  # "HiHiHi"
      
      # String methods
      text = "  Hello, World!  "
      print(text.upper())           # "  HELLO, WORLD!  "
      print(text.lower())           # "  hello, world!  "
      print(text.strip())           # "Hello, World!"
      print(text.replace("World", "Python"))  # "  Hello, Python!  "
      
      # Splitting and joining
      words = "apple,banana,cherry".split(",")
      result = "-".join(words)  # "apple-banana-cherry"
      
      # Finding and counting
      text = "Hello, World!"
      print(text.find("World"))     # 7
      print(text.count("l"))        # 3
      print(text.startswith("Hello"))  # True
      print(text.endswith("!"))     # True

9. Object-Oriented Programming

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code into objects, which contain data (attributes) and behavior (methods). Python supports OOP principles including classes, objects, inheritance, encapsulation, and polymorphism.

Classes and Objects

A class is a blueprint for creating objects. An object is an instance of a class.

# Defining a class
      class Person:
          # Class attribute
          species = "Homo sapiens"
          
          # Constructor (initializer)
          def __init__(self, name, age):
              # Instance attributes
              self.name = name
              self.age = age
          
          # Instance method
          def introduce(self):
              return f"Hi, I'm {self.name} and I'm {self.age} years old"
          
          # Another method
          def have_birthday(self):
              self.age += 1
              return f"Happy birthday! Now I'm {self.age}"
      
      # Creating objects (instances)
      person1 = Person("Alice", 25)
      person2 = Person("Bob", 30)
      
      # Accessing attributes
      print(person1.name)  # Alice
      print(person1.age)   # 25
      
      # Calling methods
      print(person1.introduce())  # Hi, I'm Alice and I'm 25 years old
      person1.have_birthday()
      print(person1.age)  # 26

Inheritance

Inheritance allows a class to inherit attributes and methods from another class:

# Parent class (base class)
      class Animal:
          def __init__(self, name):
              self.name = name
          
          def speak(self):
              return f"{self.name} makes a sound"
      
      # Child class (derived class)
      class Dog(Animal):
          def speak(self):
              return f"{self.name} barks"
      
      class Cat(Animal):
          def speak(self):
              return f"{self.name} meows"
      
      # Using inheritance
      dog = Dog("Buddy")
      cat = Cat("Whiskers")
      
      print(dog.speak())  # Buddy barks
      print(cat.speak())  # Whiskers meows

Encapsulation

Encapsulation restricts access to certain components. Python uses naming conventions:

class BankAccount:
          def __init__(self, balance):
              self.balance = balance        # Public
              self._account_number = 12345   # Protected (convention)
              self.__pin = 1234             # Private (name mangling)
          
          def deposit(self, amount):
              self.balance += amount
          
          def get_balance(self):
              return self.balance
          
          def _internal_method(self):  # Protected method
              return "Internal"
          
          def __private_method(self):  # Private method
              return "Private"
      
      account = BankAccount(1000)
      print(account.balance)        # Accessible
      print(account._account_number)  # Accessible (but convention says don't)
      # print(account.__pin)  # Error (name mangled)

Polymorphism

Polymorphism allows different classes to be used through the same interface:

class Shape:
          def area(self):
              pass
      
      class Rectangle(Shape):
          def __init__(self, width, height):
              self.width = width
              self.height = height
          
          def area(self):
              return self.width * self.height
      
      class Circle(Shape):
          def __init__(self, radius):
              self.radius = radius
          
          def area(self):
              return 3.14159 * self.radius ** 2
      
      # Polymorphism in action
      shapes = [Rectangle(5, 4), Circle(3)]
      for shape in shapes:
          print(f"Area: {shape.area()}")

Special Methods (Magic Methods)

Special methods allow you to define how objects behave with built-in operations:

class Point:
          def __init__(self, x, y):
              self.x = x
              self.y = y
          
          def __str__(self):
              return f"Point({self.x}, {self.y})"
          
          def __repr__(self):
              return f"Point({self.x}, {self.y})"
          
          def __add__(self, other):
              return Point(self.x + other.x, self.y + other.y)
          
          def __eq__(self, other):
              return self.x == other.x and self.y == other.y
      
      p1 = Point(1, 2)
      p2 = Point(3, 4)
      p3 = p1 + p2
      print(p3)  # Point(4, 6)
      print(p1 == p2)  # False

Class Methods and Static Methods

class MyClass:
          class_var = 0
          
          def __init__(self, value):
              self.instance_var = value
          
          @classmethod
          def class_method(cls):
              return f"Class variable: {cls.class_var}"
          
          @staticmethod
          def static_method():
              return "This is a static method"
          
          def instance_method(self):
              return f"Instance variable: {self.instance_var}"
      
      # Using class method
      print(MyClass.class_method())
      
      # Using static method
      print(MyClass.static_method())
      
      # Using instance method
      obj = MyClass(10)
      print(obj.instance_method())
