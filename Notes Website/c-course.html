1. Introduction to C

What is C?

C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs in 1972. It was created as an evolution of the B programming language and was designed to be a system programming language for the UNIX operating system. C is one of the most widely used programming languages and serves as the foundation for many other languages like C++, Java, Python, and JavaScript.

C is a compiled language, meaning that source code is translated into machine code by a compiler before execution. This makes C programs extremely fast and efficient. The language provides low-level access to memory through pointers, while still maintaining high-level features like structured programming constructs.

Key Characteristics of C:

  • Procedural Language: C follows a procedural programming paradigm where programs are organized as a sequence of procedures or functions
  • Statically Typed: Variable types must be declared before use and cannot be changed during runtime
  • Compiled Language: Source code is compiled to machine code, resulting in efficient execution
  • Low-level Access: Provides direct access to memory through pointers, allowing fine-grained control
  • Portable: C code can be compiled and run on different platforms with minimal modifications
  • Modular: Supports function-based modular programming for code organization

Why Learn C?

  • Foundation of Modern Programming: Understanding C helps you understand how computers work at a low level. Many modern languages are built on concepts from C
  • Performance: C is fast and efficient, making it ideal for system programming, embedded systems, and performance-critical applications
  • Portability: C programs can run on different platforms (Windows, Linux, macOS) with minimal changes, making it highly portable
  • Industry Standard: Many operating systems (Linux, Windows kernel components), databases, compilers, and applications are written in C
  • Memory Management: Learning C teaches you manual memory management, which is crucial for understanding how programs use memory
  • Career Opportunities: Knowledge of C opens doors to system programming, embedded systems, game development, and performance optimization roles
  • Understanding Other Languages: Many concepts in C++, Java, and other languages become easier to understand once you know C

Your First C Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Explanation:

  • #include <stdio.h> - Includes standard input/output library
  • int main() - Main function where program execution begins
  • printf() - Function to print text to console
  • return 0; - Indicates successful program execution

2. Variables & Data Types

What are Variables?

Variables are containers that store data values. In C, you must declare a variable before using it, specifying its data type.

Basic Data Types in C

Data Type Size (bytes) Description
int 4 Stores integers (whole numbers)
float 4 Stores floating-point numbers (decimals)
double 8 Stores large floating-point numbers
char 1 Stores single character

Example: Declaring Variables

#include <stdio.h>

int main() {
    // Integer variable
    int age = 25;
    
    // Floating-point variable
    float height = 5.9;
    
    // Character variable
    char grade = 'A';
    
    // Double variable
    double salary = 50000.50;
    
    // Displaying variables
    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);
    printf("Grade: %c\n", grade);
    printf("Salary: %.2lf\n", salary);
    
    return 0;
}

Format Specifiers

  • %d - for integers
  • %f - for floats
  • %lf - for doubles
  • %c - for characters
  • %s - for strings

3. Operators

Types of Operators

1. Arithmetic Operators

Operator Description Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus (remainder) a % b

2. Relational Operators

Operator Description
== Equal to
!= Not equal to
< Less than
> Greater than
<= Less than or equal to
>= Greater than or equal to

3. Logical Operators

Operator Description
&& Logical AND
|| Logical OR
! Logical NOT

Example: Using Operators

#include <stdio.h>

int main() {
    int a = 10, b = 5;
    
    // Arithmetic operations
    printf("Addition: %d\n", a + b);
    printf("Subtraction: %d\n", a - b);
    printf("Multiplication: %d\n", a * b);
    printf("Division: %d\n", a / b);
    printf("Modulus: %d\n", a % b);
    
    // Relational operations
    printf("a > b: %d\n", a > b);
    printf("a == b: %d\n", a == b);
    
    // Logical operations
    printf("a > 5 && b < 10: %d\n", a > 5 && b < 10);
    
    return 0;
}

4. Control Flow

If-Else Statement

The if-else statement allows you to execute different code blocks based on conditions.

Example: If-Else

#include <stdio.h>

int main() {
    int number = 10;
    
    if (number > 0) {
        printf("Number is positive\n");
    } else if (number < 0) {
        printf("Number is negative\n");
    } else {
        printf("Number is zero\n");
    }
    
    return 0;
}

Switch Statement

The switch statement allows you to select one of many code blocks to execute.

Example: Switch

#include <stdio.h>

int main() {
    int day = 3;
    
    switch (day) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        default:
            printf("Invalid day\n");
    }
    
    return 0;
}

Loops

1. For Loop

#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

2. While Loop

#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++;
    }
    return 0;
}

3. Do-While Loop

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d ", i);
        i++;
    } while (i <= 5);
    return 0;
}

5. Functions

What are Functions?

Functions are blocks of code that perform a specific task. They help in code reusability and organization.

Function Syntax

return_type function_name(parameters) {
    // function body
    return value;
}

Example: Simple Function

#include <stdio.h>

// Function declaration
int add(int a, int b);

int main() {
    int result = add(5, 3);
    printf("Sum: %d\n", result);
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Types of Functions

  • Library Functions: Predefined functions (e.g., printf, scanf)
  • User-defined Functions: Functions created by the programmer

Example: Function with No Return Value

#include <stdio.h>

void greet() {
    printf("Hello, World!\n");
}

int main() {
    greet();
    return 0;
}

6. Arrays

What are Arrays?

An array is a collection of elements of the same data type stored in contiguous memory locations.

Declaring Arrays

data_type array_name[size];

Example: Array Declaration and Usage

#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    int i;
    
    // Accessing array elements
    for (i = 0; i < 5; i++) {
        printf("numbers[%d] = %d\n", i, numbers[i]);
    }
    
    return 0;
}

Multi-dimensional Arrays

#include <stdio.h>

int main() {
    int matrix[3][3] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    int i, j;
    for (i = 0; i < 3; i++) {
        for (j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}

7. Pointers

What are Pointers?

A pointer is a variable that stores the memory address of another variable. Pointers are one of the most powerful features of C.

Pointer Syntax

data_type *pointer_name;

Example: Basic Pointer Usage

#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = #  // ptr stores address of num
    
    printf("Value of num: %d\n", num);
    printf("Address of num: %p\n", &num);
    printf("Value of ptr: %p\n", ptr);
    printf("Value pointed by ptr: %d\n", *ptr);
    
    return 0;
}

Pointer Operators

  • & - Address operator (returns address of variable)
  • * - Dereference operator (returns value at address)

Example: Pointer Arithmetic

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *ptr = arr;
    
    int i;
    for (i = 0; i < 5; i++) {
        printf("arr[%d] = %d\n", i, *(ptr + i));
    }
    
    return 0;
}

8. Strings

What are Strings?

In C, a string is an array of characters terminated by a null character ('\0').

String Declaration

char string_name[size];

Example: String Operations

#include <stdio.h>
#include <string.h>

int main() {
    char name[50] = "Hello";
    char greeting[50];
    
    // String length
    printf("Length: %lu\n", strlen(name));
    
    // String copy
    strcpy(greeting, name);
    printf("Copied string: %s\n", greeting);
    
    // String concatenation
    strcat(greeting, " World");
    printf("Concatenated: %s\n", greeting);
    
    // String comparison
    if (strcmp(name, greeting) == 0) {
        printf("Strings are equal\n");
    } else {
        printf("Strings are not equal\n");
    }
    
    return 0;
}

Common String Functions

  • strlen() - Returns length of string
  • strcpy() - Copies one string to another
  • strcat() - Concatenates two strings
  • strcmp() - Compares two strings

9. Structures

What are Structures?

A structure is a user-defined data type that allows you to combine different data types under a single name.

Structure Syntax

struct structure_name {
    data_type member1;
    data_type member2;
    // ...
};

Example: Structure Usage

#include <stdio.h>
#include <string.h>

struct Student {
    int id;
    char name[50];
    float marks;
};

int main() {
    struct Student s1;
    
    s1.id = 1;
    strcpy(s1.name, "John Doe");
    s1.marks = 85.5;
    
    printf("Student ID: %d\n", s1.id);
    printf("Student Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);
    
    return 0;
}

Array of Structures

#include <stdio.h>

struct Student {
    int id;
    char name[50];
    float marks;
};

int main() {
    struct Student students[3];
    int i;
    
    for (i = 0; i < 3; i++) {
        students[i].id = i + 1;
        printf("Enter name for student %d: ", i + 1);
        scanf("%s", students[i].name);
        printf("Enter marks: ");
        scanf("%f", &students[i].marks);
    }
    
    return 0;
}

10. File Handling

File Operations in C

C provides functions to perform operations on files: creating, reading, writing, and closing files.

File Modes

  • "r" - Read mode
  • "w" - Write mode (creates new file or overwrites existing)
  • "a" - Append mode
  • "r+" - Read and write mode

Example: Writing to a File

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w");
    
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    fprintf(file, "Hello, World!\n");
    fprintf(file, "This is a test file.\n");
    
    fclose(file);
    printf("File written successfully!\n");
    
    return 0;
}

Example: Reading from a File

#include <stdio.h>

int main() {
    FILE *file;
    char ch;
    
    file = fopen("example.txt", "r");
    
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    printf("File contents:\n");
    while ((ch = fgetc(file)) != EOF) {
        printf("%c", ch);
    }
    
    fclose(file);
    return 0;
}

Common File Functions

  • fopen() - Opens a file
  • fclose() - Closes a file
  • fprintf() - Writes formatted data to file
  • fscanf() - Reads formatted data from file
  • fgetc() - Reads a character from file
  • fputc() - Writes a character to file
